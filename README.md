# Operación Panito

Plataforma web para gestionar actividades de voluntariado, requerimientos, inscripciones de voluntarios, publicaciones de blog y administración interna.

## Stack

- Frontend: React 18 con Vite
- Backend: Node.js 20 con Express 4
- Base de datos: PostgreSQL 16

## Instalación y ejecución

Para instalar y correr el frontend:

```bash
cd frontend && npm install && npm run dev
```

Para instalar y correr el backend:

```bash
cd backend && npm install && npm run dev
```

## Puertos

- Frontend: http://localhost:5173
- Backend: http://localhost:3001

## Base de datos

La estructura inicial de PostgreSQL está disponible en:

```text
backend/db/schema.sql
```
